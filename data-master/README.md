# 2019 科技大擂台 資料處理相關程式

### date team
- CSV_to_DROP 可將方便人工編輯的csv格式題目,產出SQuAD的是否題

- FGC_to_SQuAD 分別讀入官方範例檔案題目與答案,並產出SQuAD的是否題

- SQuAD_to_DROP 將QDSQA的SQuAD格式題目，轉換成DROP格式

- fix_QDSQA 將既有QDSQA題目修正重複題目bug
